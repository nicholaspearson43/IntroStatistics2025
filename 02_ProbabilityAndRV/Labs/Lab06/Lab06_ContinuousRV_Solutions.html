Random variables

A random variable is a variable that can take on a different value based on the outcome of a random experiment. Mathematically speaking a random variable \(X\) is a real-valued function on the space of all possible outcomes \(\Omega\): \[X:\Omega\to\mathbb{R}\]

They can be categorized in two main classes: discrete and continuous.

Continuous random variables

By definition, a continuous random variable is a random variable that can take on an infinite number of values within a certain range. Each variable is defined through the probability density function (pdf) as follows: \[p(x)=Pr(a\leq X \leq b) = \int_{a}^{b} f(x)dx\]

and the following properties hold:

  • \(f(x)\geq 0\)
  • \(\int_{-\infty}^{+\infty} f(x)dx=1\)

Examples of continuous r.v. are the Gaussian (or Normal distribution), the Chi-squared distribution and the Gamma distribution.

A known theoretical result shows that the distribution of the mean \(\mu\) of a Normal distribution, is itself a Normal distribution with known parameters \[\mathcal{N}(\mu, \sigma^2/n)\]. Let’s test this with R:

R <- 1000
n <- 30
means <- vector(mode="numeric", length=R)

for (i in 1:R){
  x <- rnorm(n)
  means[i] <- mean(x)
}
mean(means)
## [1] 0.0001270866
var(means)
## [1] 0.03474316
hist(means, breaks=30, prob=TRUE)
curve(dnorm(x, mean=0, sd=sqrt(1/n)), col=2, lwd=2, add=T) #why mean = 0 and sd = sqrt(1/n) ?
